# 데이터 분석 포트폴리오

## 소개
데이터를 기반으로 문제를 정의하고, 분석을 통해 인사이트를 도출하는 데이터 분석가 지망생입니다.  
Python과 Pandas, SQL을 활용한 데이터 처리 및 분석 경험이 있습니다.

## 기술 스택
- Language: Python, SQL
- Analysis: Pandas, NumPy
- Visualization: Matplotlib, Seaborn
- Database: MongoDB
- Tools: Git, GitHub, VS Code

## 프로젝트

### 1. 고객 구매 데이터 분석
- 목적: 고객 구매 패턴 분석을 통한 매출 기여도 파악
- 주요 작업:
  - 데이터 전처리 및 EDA
  - 고객별 매출 집계
  - 가설 설정 및 검증


### 2. 데이터 전처리 파이프라인 구축
- 목적: raw 데이터를 분석 가능한 형태로 정제
- 주요 작업:
  - 결측치/이상치 처리
  - 전처리 로직 함수화
